meus projetos

refrigerante

O projeto Refrigerante é um carrossel interativo desenvolvido com HTML, CSS e JavaScript que permite ao usuário navegar entre diferentes latas de refrigerantes utilizando botões de avançar e voltar.

wineHub

Página para apresentação e venda de vinhos, desenvolvida com HTML, CSS e JavaScript. O usuário pode explorar os produtos disponíveis e realizar compras diretamente pelo site, com um visual elegante e foco na simplicidade da navegação.

projeto do tempo

Aplicação web de previsão do tempo desenvolvida com HTML, CSS e JavaScript. O usuário pode buscar qualquer cidade e visualizar informações meteorológicas em tempo real, como temperatura atual. O projeto utiliza a API da OpenWeather para fornecer dados atualizados.
